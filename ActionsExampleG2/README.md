# Actions Example

## Overview

A demonstration Android project focused on setting up and using GitHub Actions for continuous integration. The app itself is simple, but the project showcases how to configure a workflow that automatically verifies the code compiles correctly whenever changes are pushed to GitHub.

## Opening the Project in Android Studio

1. Clone or download this repository to your local machine.
2. Open Android Studio.
3. Select **File > Open** from the menu bar.
4. Navigate to the `ActionsExampleG2` folder and click **OK**.
5. Wait for Gradle to sync — this may take a few minutes on first open.
6. Once synced, you can run the app on an emulator or connected device using the **Run** button.

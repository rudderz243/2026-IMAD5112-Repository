# Basic Calculator

## Overview

A simple Android calculator app that demonstrates the use of arithmetic operators in Kotlin. The user can enter two numbers and perform addition, subtraction, multiplication, and division operations, with the result displayed on screen.

## Opening the Project in Android Studio

1. Clone or download this repository to your local machine.
2. Open Android Studio.
3. Select **File > Open** from the menu bar.
4. Navigate to the `BasicCalculatorG2` folder and click **OK**.
5. Wait for Gradle to sync — this may take a few minutes on first open.
6. Once synced, you can run the app on an emulator or connected device using the **Run** button.

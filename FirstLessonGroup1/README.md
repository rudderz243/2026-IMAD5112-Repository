# First Lesson

## Overview

This is the very first Android app built in class. It serves as an introduction to Android Studio and the Kotlin programming language, covering how to create a basic app, set up a project, and understand the structure of an Android application.

## Opening the Project in Android Studio

1. Clone or download this repository to your local machine.
2. Open Android Studio.
3. Select **File > Open** from the menu bar.
4. Navigate to the `FirstLessonGroup1` folder and click **OK**.
5. Wait for Gradle to sync — this may take a few minutes on first open.
6. Once synced, you can run the app on an emulator or connected device using the **Run** button.

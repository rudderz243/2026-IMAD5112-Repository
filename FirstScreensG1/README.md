# First Screens

## Overview

An Android app that introduces multi-screen navigation. The project covers how to create multiple activities, navigate between them using Intents, and structure an app with more than one screen — a fundamental skill in Android development.

## Opening the Project in Android Studio

1. Clone or download this repository to your local machine.
2. Open Android Studio.
3. Select **File > Open** from the menu bar.
4. Navigate to the `FirstScreensG1` folder and click **OK**.
5. Wait for Gradle to sync — this may take a few minutes on first open.
6. Once synced, you can run the app on an emulator or connected device using the **Run** button.

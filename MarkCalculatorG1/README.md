# Mark Calculator

## Overview

An Android app that calculates a student's final mark based on inputs such as assignment scores and test results. It applies conditional logic and arithmetic to compute a percentage and determine a grade, integrating several concepts covered throughout the course.

## Opening the Project in Android Studio

1. Clone or download this repository to your local machine.
2. Open Android Studio.
3. Select **File > Open** from the menu bar.
4. Navigate to the `MarkCalculatorG1` folder and click **OK**.
5. Wait for Gradle to sync — this may take a few minutes on first open.
6. Once synced, you can run the app on an emulator or connected device using the **Run** button.

# Math Test

## Overview

A three-screen Android app that functions as an interactive math quiz. The app presents arithmetic questions across multiple screens, collects user answers, and displays a final score — bringing together multi-screen navigation, data passing, loops, and conditional logic into a single cohesive project.

## Opening the Project in Android Studio

1. Clone or download this repository to your local machine.
2. Open Android Studio.
3. Select **File > Open** from the menu bar.
4. Navigate to the `MathTestG2` folder and click **OK**.
5. Wait for Gradle to sync — this may take a few minutes on first open.
6. Once synced, you can run the app on an emulator or connected device using the **Run** button.

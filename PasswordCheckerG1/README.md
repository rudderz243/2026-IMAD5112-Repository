# Password Checker

## Overview

An Android app that evaluates a password entered by the user and determines whether it meets certain strength criteria. It uses `if-else` logic to check conditions such as length and character variety, providing feedback on whether the password is acceptable.

## Opening the Project in Android Studio

1. Clone or download this repository to your local machine.
2. Open Android Studio.
3. Select **File > Open** from the menu bar.
4. Navigate to the `PasswordCheckerG1` folder and click **OK**.
5. Wait for Gradle to sync — this may take a few minutes on first open.
6. Once synced, you can run the app on an emulator or connected device using the **Run** button.

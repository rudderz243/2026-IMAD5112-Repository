# Password Guesser

## Overview

An Android app that simulates a password guessing game. The user attempts to guess a preset password, and the app uses conditional logic to inform them whether their guess is correct or not. This project reinforces `if-else` thinking in an interactive context.

## Opening the Project in Android Studio

1. Clone or download this repository to your local machine.
2. Open Android Studio.
3. Select **File > Open** from the menu bar.
4. Navigate to the `PasswordGuesserG2` folder and click **OK**.
5. Wait for Gradle to sync — this may take a few minutes on first open.
6. Once synced, you can run the app on an emulator or connected device using the **Run** button.

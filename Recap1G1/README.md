# Recap 1

## Overview

A recap Android project focused on multi-screen navigation and data passing. The app features three screens: a Welcome screen, an Input screen to add item names and counts (up to a limit of 10), and an Output screen that fetches and displays the saved items using intents and loops.

## Opening the Project in Android Studio

1. Clone or download this repository to your local machine.
2. Open Android Studio.
3. Select **File > Open** from the menu bar.
4. Navigate to the `Recap1G1` folder and click **OK**.
5. Wait for Gradle to sync — this may take a few minutes on first open.
6. Once synced, you can run the app on an emulator or connected device using the **Run** button.

# Recap 2

## Overview

A comprehensive recap Android project that combines multi-screen navigation, fixed-size arrays, data passing, custom functions, and complex loop logic. The app has three screens: a Welcome screen, a student mark entry screen (up to 10 entries with validation), and a summary screen that calculates and displays the full class list, the class average using a custom Kotlin function, and the highest and lowest marks.

## Opening the Project in Android Studio

1. Clone or download this repository to your local machine.
2. Open Android Studio.
3. Select **File > Open** from the menu bar.
4. Navigate to the `Recap2G2` folder and click **OK**.
5. Wait for Gradle to sync — this may take a few minutes on first open.
6. Once synced, you can run the app on an emulator or connected device using the **Run** button.

# Submission Recap

## Overview

A recap Android project that consolidates the skills learned so far, including GitHub version control, GitHub Actions for automated builds, and the practice of writing a clear and structured README. This project reinforces good development habits around project documentation and source control.

## Opening the Project in Android Studio

1. Clone or download this repository to your local machine.
2. Open Android Studio.
3. Select **File > Open** from the menu bar.
4. Navigate to the `SubmissionRecapG1` folder and click **OK**.
5. Wait for Gradle to sync — this may take a few minutes on first open.
6. Once synced, you can run the app on an emulator or connected device using the **Run** button.

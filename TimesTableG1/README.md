# Times Table

## Overview

An Android app that generates and displays a times table for any number entered by the user. This project introduces loop structures in Kotlin — specifically `for` and `while` loops — to repeatedly perform calculations and build up a list of results.

## Opening the Project in Android Studio

1. Clone or download this repository to your local machine.
2. Open Android Studio.
3. Select **File > Open** from the menu bar.
4. Navigate to the `TimesTableG1` folder and click **OK**.
5. Wait for Gradle to sync — this may take a few minutes on first open.
6. Once synced, you can run the app on an emulator or connected device using the **Run** button.
